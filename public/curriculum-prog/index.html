ラズベリーパイ

小学生プログラミングコースではRaspberryPi400を使用し下記のような項目を学んでいきます。またRaspberryPi400はオリジナルのRaspberryPiシングルボードコンピューターと同様に、LEDライト、ボタン、モーター、およびあらゆる種類の電子部品を制御するためのGPIOヘッダーがあります。

CSアンプラグド（CS：コンピュターサイエンス）

CSアンプラグドとは「コンピュターを使用しない」アクティビティです。例えばアルゴリズム、エンコーディング、エラー検出、条件などのコンピュータサイエンスとプログラミングの基礎は、カードゲームやピクセル描画プロジェクトなどの「コンピュターを使用しない」アクティビティ中に取得できます。中級または上級者も、「コンピュターを使用しない」アクティビティを通じて、コンピュータベースの「プラグイン」アクティビティでフォローアップすることができます。

Scratch

Scratchは、すべての年齢の子供と初心者のために構築されたビジュアルブロックプログラミング言語です。コードの結果を視覚的かつ即座に確認します。

プロジェクトベース

明確な目標に向かって取り組み、慎重なタスクを完了するために必要なツールを収集します。 Scratchの「Hello, World!」プログラムからゲームやアニメーションまで、学生には常に明確な目標を設定し完了させていきます。

クリエイティブコーディング

創造的なコーディング活動は、子供たちがコンピューター通して自分自身を表現することを可能にします。子供たちの想像力を使い、独自のゲーム、クイズ、アニメーションなどを作成することができます。作ったものを友達や家族と簡単に共有できます。

Linux　/ ターミナル

中級および上級者へは、Webの基礎となるオペレーティングシステムとそのテキストベースのインターフェイスに精通していることは、必須のスキルです。

Python

ビジュアルブロック(Scratch)を習得した後、Python言語を使用したテキストベースのプログラミングに進むことができます。 Pythonにはシンプルな構文がありますが、様々な場面で活用できる強力なツールでもあります。

Sonic Pi

Sonic Piはオープンソースのプログラミング環境で、音を作りながらプログラミングの概念を教えたり探索したりするようにデザインされたものです。それは、ケンブリッジ大学のコンピュータ・ラボラトリーのSam Aaronによって作られた、誰でも使える無料のライブコーディング・シンセです。Sonic Piを使って、カノンのようなクラシックからダブステップのような現代的なスタイルまで幅広い音楽をプログラミングして作曲することができます。
